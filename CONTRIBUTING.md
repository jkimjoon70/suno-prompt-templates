# Contributing

Thank you for your interest in contributing to this project.

This repository is an open-source Suno prompt template library for non-developers, AI music creators, educators, and Korean/English content creators.

## How you can contribute

You can contribute by:

- Suggesting new Suno prompt templates
- Improving Korean or English wording
- Adding genre-specific examples
- Sharing AI music workflow ideas
- Improving documentation
- Reporting unclear or outdated prompt structures

## Template Format

When adding a new template, please use the following structure:

```markdown
# Template Name

## Concept

Describe the overall song concept.

## Lyrics Direction

- Theme:
- Language:
- Vocal style:
- Mood:

## Style Prompt

Paste the Suno style prompt here.

## Recommended Use Case

Explain when this template is useful.
```

## Contribution Guidelines

Please make sure that your contribution is:

- Clear and easy to understand
- Useful for AI music creators
- Safe for public use
- Written in Korean, English, or both
- Not copied from copyrighted private prompt collections
