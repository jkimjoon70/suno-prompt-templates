# 🎵 Suno Prompt Templates

An open-source Suno prompt template library for non-developers, AI music creators, educators, and Korean/English content creators.

## Purpose

This project helps creators design music concepts, lyrics structures, genre-specific prompts, and reusable AI music workflows without requiring coding skills.

이 프로젝트는 코딩 지식이 없어도 음악 콘셉트, 가사 구조, 장르별 프롬프트, AI 음악 제작 워크플로우를 쉽게 설계할 수 있도록 돕습니다.

---

## Who is this for?

- Non-developer AI music creators
- Korean creators using Suno
- Educators teaching AI content creation
- Small creative teams building album concepts and lyrics workflows
- Creators who need reusable Korean/English prompt templates
- Independent artists building full album concepts with AI tools

---

## What is this?

A curated, battle-tested collection of Suno AI prompt templates built from real album production experience.

These templates were developed and refined through the production of:

- **"떨림을 넘어서" (Beyond the Tremor)** — 48-track Parkinson's Disease symphonic metal album series
- **"치유의 불꽃" (Flame of Healing)** — 20-track healing meditation album
- **"K-Geo Symphonic Universe"** — 20-track geographic metal series based on Gwangju mountains

All templates are optimized for **Suno v5.5 with Dynamic Mode**.

---

## Quick Start

Choose a template, copy the prompt structure, and customize the concept, language, vocal style, genre, and mood.

Recommended workflow:

1. Select a genre template
2. Define the song concept
3. Write or generate lyrics
4. Copy the style prompt into Suno
5. Adjust the result through iteration

## Template Structure

Each template can be organized with the following structure:

```markdown
# Template Name

## Concept

Describe the overall song concept.

## Lyrics Direction

- Theme:
- Language:
- Vocal style:
- Mood:

## Style Prompt

Paste the Suno style prompt here.

## Recommended Use Case

Explain when this template is useful.
```

## Korean Guide

### 이 저장소는 무엇인가요?

이 저장소는 실제 앨범 제작 경험을 바탕으로 정리한 **Suno AI 프롬프트 템플릿 모음**입니다.

단순한 문장 모음이 아니라, 실제 AI 음악 제작 과정에서 반복적으로 테스트하고 다듬은 프롬프트 구조를 제공합니다.

이 템플릿들은 다음과 같은 작업을 통해 개발되고 정리되었습니다.

- **"떨림을 넘어서"** — 파킨슨병 인식 개선을 위한 48트랙 심포닉 메탈 앨범 시리즈
- **"치유의 불꽃"** — 20트랙 치유 명상 앨범
- **"K-Geo Symphonic Universe"** — 광주 지역 산을 바탕으로 한 20트랙 지리 메탈 시리즈

모든 템플릿은 **Suno v5.5 Dynamic Mode** 사용을 기준으로 최적화되어 있습니다.

### 사용 방법

1. 원하는 장르 템플릿을 고릅니다.

5. 결과물을 듣고 단어와 분위기를 조정합니다.

## Why this project matters

AI music tools are powerful, but many creators do not know how to structure prompts clearly.

This repository lowers the barrier for non-developers and independent creators by providing reusable, practical, and adaptable prompt templates.

이 프로젝트는 코딩을 모르는 창작자도 AI 음악 제작에 쉽게 접근할 수 있도록 돕습니다.

특히 한국어 창작자, 1인 창작자, 강사, 교육자, 소규모 창작팀이 Suno를 활용해 음악 콘셉트, 가사 구조, 장르별 프롬프트를 체계적으로 만들 수 있도록 지원합니다.
